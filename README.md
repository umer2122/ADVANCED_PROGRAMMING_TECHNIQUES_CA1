## 🧾 Tasks Overview

### Task 1 – Churros Ordering System (C#)
- Menu-driven console application  
- Implements Object-Oriented Programming (OOP)  
- Uses Queue data structure (FIFO)  
- Features: order placement, billing, and delivery  
- Includes unit testing for bill calculation  

---

### Task 2 – Periodic Table System (C#)
- Stores first 30 elements of the periodic table  
- Uses Dictionary for efficient data retrieval  
- User inputs atomic number to get element details  
- Includes input validation and interactive loop  

---

### Task 3 – EasyDrive Client-Server Application (Python)
- TCP-based client-server system using sockets  
- Client collects user registration data  
- Server processes and stores data in SQLite database  
- Generates unique registration number  
- Demonstrates multithreading and database integration  

---

### Task 4 – Web Scraping (Python)
- Scrapes book data (name, rating, price) from website  
- Uses `requests` and `BeautifulSoup`  
- Stores extracted data in CSV file  
- Reads and displays data from CSV  
- Handles encoding issues for correct output
